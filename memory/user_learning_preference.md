---
name: User implementation preference
description: Whether the user wants to be guided/taught or have code implemented directly for them
type: user
---

User initially wanted to learn programming by doing — explained and guided, not having code written for them. However, when facing large multi-feature implementation tasks, they explicitly prefer having everything implemented directly without explanations ("Pode implementar tudo! Sem me explicar!").

**How to apply:** For single features or learning moments, guide the user. For large scope tasks (many features at once), implement directly without step-by-step teaching unless they ask for it.
